## ğŸ¨ Customization

### Update Content
- Edit `app/page.js` to modify personal information, projects, and experiences
- Update metadata in `app/layout.js` for SEO

### Add Your Resume
- Place your CV PDF in the `public` folder as `Mark_CV_2025.pdf`
- Update the download link in the Contact section if needed

### Modify Theme
- Adjust colors in `tailwind.config.js`
- Modify animations in `app/globals.css`
- Update fonts in the Google Fonts import
